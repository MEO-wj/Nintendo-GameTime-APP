# Nintendo GameTime — 产品项目介绍

> 一款面向 Nintendo Switch 玩家的游戏库管理与游玩时长追踪应用

---

## 一、产品概述

Nintendo GameTime 是一个「游戏墙 + 游玩时长账本」式的 Web 应用，帮助玩家系统化管理自己的 Switch 游戏收藏与游玩数据。它通过对接 Nintendo 官方 API 实时同步游戏列表和时长数据，并提供手动修正机制来弥补官方数据的缺失或不准确之处。

### 核心价值

- **游戏收藏可视化** — 以游戏墙形式展示全部已购游戏，附带封面、价格、评分等元信息
- **游玩时长追踪** — 自动同步官方时长数据，支持手动校准，确保数据真实可靠
- **数据洞察仪表盘** — 通过环形图、柱状图、矩形树图等可视化手段呈现游玩分布
- **双端覆盖** — Web 浏览器 + 移动端原生壳（Capacitor），随时随地查看

### 目标用户

主要面向中国大陆 Nintendo Switch 玩家，UI 以简体中文为主，同时支持「国内 / 海外」双模式切换。

---

## 二、功能特性

### 1. 用户认证

| 特性 | 说明 |
|------|------|
| 登录方式 | 邮箱 + OTP 验证码 |
| 会话管理 | JWT Token（7 天有效期） |
| 开发模式 | 任意邮箱 + 固定验证码 `000000` |

### 2. Nintendo 账号绑定

- 粘贴 Nintendo Session Token 完成绑定
- Token 经 AES-256-GCM 加密后存储，保障安全性
- 绑定后自动触发首次数据同步

### 3. 游戏数据同步

- 通过 Nintendo 内部 API (`api-lp1.znc.srv.nintendo.net`) 获取游戏列表与游玩时长
- 后台 Worker 每 5 分钟自动轮询同步（间隔可配置）
- 支持 Mock 模式用于开发调试
- 同步失败追踪与告警机制

### 4. 游戏目录

- 预置约 300 款 Nintendo Switch 热门游戏
- 自动抓取 Nintendo eShop 页面获取元数据（封面、价格、描述）
- 支持中文本地化覆盖（20+ 款重点游戏提供中文标题）
- 背景数据补全机制，保证首次加载速度

### 5. 游戏库管理

- 从目录添加游戏到个人库
- 三种视图模式：按入手时间 / 按最近游玩 / 按游玩时长排序
- 支持移除游戏（软删除）

### 6. 评分系统

- 玩家自定义评分（0.1 ~ 10.0 精度）
- 汇总平均分与评分人数
- 展示 Metacritic 风格的媒体评分

### 7. 仪表盘与数据可视化

- **概览统计**：游戏总数、总游玩时长、游戏总价值、近 30 天活跃度
- **环形图**：Top 5 游戏时长分布
- **柱状图**：全部游戏时长排名
- **矩形树图**：游戏库按可视化时长分布
- 可选 R 语言驱动的高级图表生成

### 8. 显示偏好

| 模式 | 效果 |
|------|------|
| 国内模式 (DOMESTIC) | 中文游戏标题、人民币价格（基于 ECB 汇率换算） |
| 海外模式 (GLOBAL) | 英文游戏标题、原始货币价格 |

---

### 9、设计理念

1. **不覆盖原则** — 修正账本永远保留官方原始快照，所有修正都是可追溯、可撤销的叠加操作
2. **渐进增强** — 游戏目录先返回缓存数据，后台异步补全详细元信息
3. **安全优先** — Nintendo Session Token 使用 AES-256-GCM 加密存储，JWT 有过期机制
4. **开发友好** — 内存存储模式 + Mock 数据，零外部依赖即可启动开发
5. **双模显示** — 同时服务国内和海外玩家，汇率数据基于 ECB 官方源
